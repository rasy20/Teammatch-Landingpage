# 🚀 TeamMatch — Landing Page

TeamMatch adalah platform pencarian tim lomba mahasiswa berbasis skill, track record, dan testimoni. Project ini berupa landing page yang dirancang untuk memperkenalkan konsep aplikasi sekaligus menarik user untuk bergabung.

Landing page ini mengangkat permasalahan umum di kalangan mahasiswa, seperti sulitnya mencari rekan tim yang cocok, seringnya terjadi ghosting dalam proses rekrut, serta tidak adanya sistem validasi kemampuan antar anggota tim. Melalui pendekatan visual yang bold dengan gaya **neobrutalism**, halaman ini menekankan kecepatan, kejelasan, dan pengalaman yang kuat bagi pengguna.
